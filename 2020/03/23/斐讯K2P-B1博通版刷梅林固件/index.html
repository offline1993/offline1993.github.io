斐讯K2P B1博通版刷梅林固件

发表于 2020-03-23 更新于 2021-02-18 分类于 折腾 阅读次数：

固件下载

K2P 的梅林固件比较老了，好几年没有更新，直接贴在这吧：

K2P B1 博通版 梅林固件 1.2

固件安装

如果下载的是压缩包，请 解压出来刷入后缀名为 trx 的文件

if 当前是梅林固件

如果你现在已经是梅林固件了，直接到路由器设置页面升级：系统管理 - 固件升级 - 上传新固件

if 当前是官方或官改固件
刷入步骤

  1. 将计算机的以太网设置为固定 IP ：192.168.2.2（如果你的路由器IP是 192.168.2.1，如不是请修改）

  2. 拔掉路由器的电源，进入关机状态

  3. 将计算机连接路由器的 LAN 口

  4. 按住路由器的复位键，插上电源，等待 10 秒左右松开复位键

  5. 访问 http://192.168.2.1 ，确认可以正确进入 CFE 界面

  6. 下载 tftp 服务器工具，将 trx 固件文件放入与 tftp 服务器工具同目录（tftp 工具存放路径不要有中文字符）

  7. 打开 tftp ，选择 192.168.2.2 的网卡

  8. 打开浏览器，输入

    http://192.168.2.1/do.htm?cmd=flash+-noheader+192.168.2.2:K2P_Merlin_V12.trx+flash0.trx （其中 K2P_Merlin_V12.trx 是要刷入固件的文件名，如不是请修改）

  9. 此时浏览器会提示 未找到页面，不要刷新页面！！

  10. 查看 tfpt 的发送文件进度，等待 5 分钟左右刷机完成，此时可以用以下命令来查看刷机状态，刷机成功后就可以收到数据包了：

    ping 192.168.2.1 -t

成功后操作：
  1. 进入 CFE 界面清除 NVRAM：浏览器访问 http://192.168.2.1/do.htm?cmd=nvram+erase
  2. 进入 CFE 界面重启路由器（或断电重启）
  3. 恢复出厂设置
  4. 路由器的设置地址会变为：http://192.168.1.1

注意事项

同网段操作

保证以下 IP 均在同一网段：

  • 计算机 IP
  • 路由器 网关 IP
  • CEF 下网址命令的 IP

比如 路由器 LAN IP 为 192.168.2.1，那么计算机的 IP 应该为 192.168.2.2 ，CEF 下网址命令的 IP 应该为 192.168.2.1

不要在线升级！

梅林固件设置界面可能会出现惊叹号的新版本提示，千万不要在线更新华硕固件，否则必砖！！
